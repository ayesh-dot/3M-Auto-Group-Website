
# 3M Auto Group - Website | 🚗 Project Overview
#### This project is a custom-built web platform designed for a local car dealership. It features a sleek, "Glassmorphism" UI and a functional administrative backend to manage vehicle listings in real-time.
---

# 🛠️ Tech Stack

### Languages: HTML5, CSS3, JavaScript (ES6+)

### Database & Hosting: Firebase (Firestore & Storage)

### Typography: Gelasio (Serif) & Inter (Sans-Serif)

### Architecture: Modular CSS with more than 1,500 lines of custom code.

---

#  🚀 Key Features
### Glassmorphism UI: A custom-engineered visual interface featuring real-time background blur and sophisticated button interactions.

### Real-time Database: Fully integrated with Firebase Firestore for instant inventory updates.

### Admin Control Panel: A secure interface to add, delete, and modify vehicle listings.

### Responsive Specs Grid: Vehicle data (Mileage, Fuel, Transmission) displayed in a clean, high-readability grid.

### Automated Transitions: Smooth CSS animations for social media links and dashboard navigation.

🔒 Security & Data Integrity



*   Cloud Protection: Implemented specialized data access rules to ensure only authorized personnel can modify vehicle inventory.

*   Data Validation: Every entry field is screened to prevent database errors and ensure a consistent user experience.

*   Layout Stability: Utilizes a custom structural spacing system to maintain perfect visual balance across different screen heights.
